# BLive v0.2.3

## 更新亮点

- 弹幕密度优化：弹幕轨道由固定数量改为按字号自适应，小字号时同屏弹幕更密集。
- 播放页重构：抽离播放流解析模块与设置面板控制器，降低页面耦合并提升后续扩展性。
- 登录页重构：登录流程迁移到 ViewModel + Repository，轮询与倒计时逻辑更稳定。
- 主页面重构：列表状态、焦点导航、UI 渲染职责拆分，页面结构更清晰。
- 交互体验优化：统一纯文字 Toast 提示；主页面支持双击返回退出应用。
- 列表体验优化：直播间列表状态化，移除 mock 占位逻辑。
